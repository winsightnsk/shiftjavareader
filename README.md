# Утилита фильтрации данных из файлов
## Описание

Утилита предназначена для фильтрации содержимого текстовых файлов, содержащих целые числа, вещественные числа и строки. 
Программа распределяет данные по типам в соответствующие выходные файлы и предоставляет статистику обработки.

*Выполнена по заданию команды ШИФТ*

**Java версия:** `21` 

**Сторонние библиотеки:** `не используются`

**Сборка проекта**\
Для сборки проекта в исполняемый JAR-файл выполните:

``` bash
chmod +x compile.sh
./compile.sh
```
После выполнения команды будет создан файл shiftjavareader.jar`

## Использование
**Синтаксис команды:**

``` bash
java -jar shiftjavareader.jar [опции] файл1 файл2 ... файлN
java -jar shiftjavareader.jar --help // для вывода справочной информации
```

Опции:
`-o` <путь> - путь для сохранения выходных файлов (по умолчанию: текущая директория)

`-p` <префикс> - префикс имен выходных файлов (по умолчанию: без префикса)

`-a` - режим добавления к существующим файлам (по умолчанию: перезапись)

`-s` - вывод краткой статистики (количество элементов)

`-f` - вывод полной статистики (количество, мин/макс, сумма, среднее для чисел; длина строк для строк)

**Примеры использования:**

bash
java -jar shiftjavareader.jar -f -p result_ src/test/resources/in1.txt src/test/resources/in2.txt
